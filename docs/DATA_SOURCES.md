# Data Sources

Phase 1 rule: every meaningful claim needs a source.

## Source types to prioritize

- PEN America
- ALA / library sources
- school district records when available
- reputable local news
- publisher or author statements when directly relevant

## Data quality rules

- prefer primary or strong secondary sources
- keep source URLs on every event
- note uncertainty clearly
- separate challenged vs banned if possible
- avoid overstating claims

## Phase 1 note

The first version does not need complete national coverage.
It needs a credible, useful seed dataset.
